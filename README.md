# Nerf-Dart Book
I started this project because I really like the Alex Rider series—especially Point Blank—and the gadgets in the books are just really fun ideas. The stun dart book was always my favorite gadget, so I wanted to try making my own version of it for the Hack Club's Jackpot! Hack-a-thon.

The idea is a fake spy book that can launch foam darts (like Nerf darts). It’s meant to look like a regular hardcover book but function as a simple, harmless gadget-inspired toy. Right now it’s still a work in progress. I’m working on the design and figuring out a safe way to build the internal mechanism.
This is just a fun hardware/maker project inspired by fiction, not meant to be used as a weapon or for anything unsafe.
